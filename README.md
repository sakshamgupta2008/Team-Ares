# PreventIQ — Developer Setup Guide

## Step 1: Run the Notebook (one time)
Open `PreventiQModel_Final.ipynb` and run all cells top-to-bottom.

This will:
- Train the best ML model (XGBoost, Random Forest, etc.)
- Precompute the full risk JSON for every state and district in India
- Save `preventiq_cache.json`, `models/best_model.joblib`, `models/scaler.joblib`

**You only need to do this once.** After that, the backend just reads the cache.

---

## Step 2: Folder Structure
```
/your-project/
  preventiq_backend.py
  preventiq_cache.json        ← generated by notebook
  final_health_data.xlsx
  requirements.txt
  models/
    best_model.joblib
    scaler.joblib
    model_meta.json
```

---

## Step 3: Install & Run
```bash
pip install -r requirements.txt
uvicorn preventiq_backend:app --reload --port 8000
```
Interactive docs: http://localhost:8000/docs

---

## API Endpoints

### POST /predict  ← Main endpoint
```bash
# District level
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"state": "Madhya Pradesh", "district": "Indore"}'

# State level
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"state": "Maharashtra"}'
```

### GET /states
All states ranked by risk percentage.

### GET /districts/{state_name}
All districts in a state ranked by risk.

### GET /leaderboard
All states with rank, risk %, equity score, top driver.

### GET /health
Server status.
