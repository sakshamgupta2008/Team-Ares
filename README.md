What the Developer Needs to Do Next
Once your code is in the repo, tell your developer to pull the branch and run these two commands on their machine:

Install your dependencies:

Bash
pip install -r requirements.txt
Start your server:

Bash
python main.py
Once they do that, the backend will be running on their localhost, and they can immediately start sending patient and district data to your /api/v1/analyze-risk endpoint!
